Proyectos Tecnológicos de Elvis Da Silva

Sistema de Recepción de Llamadas con Raspberry Pi y Flask

Un sistema de telecomunicación basado en IP utilizando una Raspberry Pi con botones físicos que envían señales HTTP a un servidor Flask, que luego notifica a un cliente para aceptar la llamada.

Componentes: Raspberry Pi con GPIO y Flask, Cliente con interfaz visual para aceptar llamadas, Servidor que recibe y distribuye solicitudes.

Sistema de Monitoreo con Zabbix

Implementación de Zabbix para monitoreo de redes y servidores con configuraciones específicas para notificaciones y visualización en cuatro pantallas.

Tecnologías: Zabbix Server, Configuración de métricas y alertas personalizadas, Integración con otros sistemas de monitoreo.

Aplicación de Delivery con Flutter y MariaDB

Desarrollo de una app de delivery con seguimiento en tiempo real del motorizado y gestión de pedidos mediante una base de datos en MariaDB.

Características: Tres roles: Cliente, Administrador y Delivery, Seguimiento del motorizado en OpenStreetMap, Integración con un sistema de pagos.

Tecnologías: Flutter, MariaDB, OpenStreetMap.

Sistema de VPN con Voz Similar a Radmin VPN

Proyecto para crear una VPN con comunicación por voz integrada, ofreciendo una solución segura y fácil de usar.

Características: Implementación simple con SSL/TLS, Opción avanzada con interfaz gráfica, redirección de tráfico y comunicación por voz.

Tecnologías: Python, OpenVPN o WireGuard.

Configuración de OwnCloud en Ubuntu

Instalación y configuración de OwnCloud en un servidor Ubuntu para gestión de archivos personales y profesionales.

Tecnologías: OwnCloud, Servidor Ubuntu, Configuración de acceso remoto mediante SSH y VNC.

Tecnologías: Integración de sistemas de control de acceso, Automatización con scripts en Python y Bash, Monitoreo con Zabbix.

Autoclicker con Interfaz Escalable y Elegante
